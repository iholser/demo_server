##install:
`yarn`

##setup:
in one terminal, run:
`docker-compose up`
in another, run:
`yarn seed`

this will setup the database and fill in a couple of records. See seed_data.js

##run:
in one termina, run:
`docker-compose up`
in another, run:
`yarn start`
